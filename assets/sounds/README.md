# Ambiances sonores

Fichiers attendus (vos formats actuels) :


| Fichier     | Ambiance |
| ----------- | -------- |
| `pluie.m4a` | Pluie    |
| `foret.m4a` | Forêt    |
| `cafe.wav`  | Café     |
| `feu.wav`   | Feu      |


Après modification, reconstruisez l’APK : `npm run android:apk`

**Note :** les fichiers très lourds (ex. `foret.wav` ~30 Mo) peuvent mettre 1–2 s à démarrer. Pour l’APK, des boucles de 30–90 s en MP3/M4A compressés sont recommandées.
